## Desired Data Input

Choose one of the following input methods:

- **LinkedIn URL** (Outputs the best results)
- **Full Name + Company Domain and/or Company Name**

<Note>
Using the company domain produces more accurate results than using the company name, since domains are unique while names can have duplicates.
</Note>
